数值统计

问题描述 :

统计给定的n个数中，负数、零和正数的个数。

输入:

输入数据有多组，每组占一行，每行的第一个数是整数n（n<100），表示需要统计的数值的个数，然后是n个实数；如果n=0，则表示输入结束，该行不做处理。

输出:

输入数据有多组，每组占一行，每行的第一个数是整数n（n<100），表示需要统计的数值的个数，然后是n个实数；如果n=0，则表示输入结束，该行不做处理。

样例输入:

6 0 1 2 3 -1 0
5 1 2 3 4 0.5
0

样例输出:

1 2 3
0 0 5

题目链接：http://acm.hdu.edu.cn/showproblem.php?pid=2008

#include <stdio.h>

int main(void)
{
    int n, i, a, b, c;
    double x;

    while (scanf("%d", &n) , n)
    {
        a = b = c = 0;
        for (i = 0 ; i < n ; i++)
        {
            scanf("%lf", &x);
            if (x > 0) c++;
            else if (x < 0) a++;
            else b++;
        }
        printf("%d %d %d\n", a, b, c);
    }

    return 0;
}
