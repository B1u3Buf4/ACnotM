Secret Number

问题描述 :

有一天, KIKI 收到一张奇怪的信, 信上要KIKI 计算出给定数各个位上数字为偶数的和.
eg. 5548
结果为12 , 等于 4 + 8

KIKI 很苦恼. 想请你帮忙解决这个问题.

输入:

输入数据有多组,每组占一行,只有一个数字,保证数字在INT范围内.

输出:

输入数据有多组,每组占一行,只有一个数字,保证数字在INT范围内.

样例输入:

415326
3262

样例输出:

12

10

2011-12-16 04:29:08

地址：http://acm.hdu.edu.cn/showproblem.php?pid=2113

题意：中文。

代码：

# include <stdio.h>


int calc(int n)
{
    int sum = 0 ;
    while(n)
    {
        if ((n&1) == 0) 
            sum += (n%10) ;
        n/=10 ;
    }
    return sum ;
}


int main ()
{
    int n , flag = 0 ;
    while (~scanf ("%d", &n))
    {
        if (flag == 0) flag = 1 ;
        else puts("") ;
        printf ("%d\n", calc(n)) ;
    }
    return 0 ;
}
