A1 = ?

问题描述 :

有如下方程：Ai = (Ai-1 + Ai+1)/2 – Ci (i = 1, 2, 3, …. n).
若给出A0, An+1, 和 C1, C2, …..Cn.
请编程计算A1 = ?

输入:

输入包括多个测试实例。
对于每个实例，首先是一个正整数n,(n <= 3000); 然后是2个数a0, an+1.接下来的n行每行有一个数ci(i = 1, ….n);输入以文件结束符结束。

输出:

输入包括多个测试实例。
对于每个实例，首先是一个正整数n,(n <= 3000); 然后是2个数a0, an+1.接下来的n行每行有一个数ci(i = 1, ….n);输入以文件结束符结束。

样例输入:

1
50.00
25.00
10.00
2
50.00
25.00
10.00
20.00

样例输出:

27.50
15.00

2011-12-20 04:48:49
# include <stdio.h>
# include <string.h>


typedef struct NODE{
    double a0, a1 ;
    double c ;
}NODE ;


NODE node[3010] ;
double c[3010] ;
int n ;


double calc(double a, double b)
{
    int i ;
    memset(node, 0, sizeof(node)) ;
    node[0].a0 = node[1].a1 = 1 ;
    for (i = 2 ; i <= n+1 ; i++)
    {
        node[i].a0 = 2*node[i-1].a0 - node[i-2].a0 ;
        node[i].a1 = 2*node[i-1].a1 - node[i-2].a1 ;
        node[i].c = 2 * node[i-1].c - node[i-2].c + 2*c[i-1] ;
    }
    return (b - a*node[n+1].a0 - node[n+1].c)/node[n+1].a1 ;
}


int main ()
{
    int i ;
    double a, b ;
    while (~scanf ("%d", &n))
    {
        scanf ("%lf%lf", &a, &b) ;
        for (i = 1 ; i<= n ; i++)
            scanf ("%lf", &c[i]) ;
        printf ("%.2lf\n", calc(a,b)) ;
    }
    return 0 ;
}
