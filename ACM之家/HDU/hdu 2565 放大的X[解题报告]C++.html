放大的X

问题描述 :

请你编程画一个放大的’X’。
如3*3的’X’应如下所示： 

X X
 X
X X

5*5的’X’如下所示：
X   X
 X X
  X
 X X
X   X

输入:

输入数据第一行是一个整数T，表示有T组测试数据；
接下来有T行，每行有一个正奇数n（3 <= n <= 79），表示放大的规格。

输出:

输入数据第一行是一个整数T，表示有T组测试数据；
接下来有T行，每行有一个正奇数n（3 <= n <= 79），表示放大的规格。

样例输入:

2
3
5

样例输出:

X X
 X
X X

X   X
 X X
  X
 X X
X   X

题目： http://acm.hdu.edu.cn/showproblem.php?pid=2565
