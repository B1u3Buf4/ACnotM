九度-1522-包含min函数的栈[解题代码]

coder九度OJ围观1310次25 条评论
题目描述：

定义栈的数据结构，请在该类型中实现一个能够得到栈最小元素的min函数。

输入：

输入可能包含多个测试样例，输入以EOF结束。
对于每个测试案例，输入的第一行为一个整数n(1<=n<=1000000)， n代表将要输入的操作的步骤数。
接下来有n行，每行开始有一个字母Ci。
Ci=’s’时，接下有一个数字k，代表将k压入栈。
Ci=’o’时，弹出栈顶元素。

输出：

对应每个测试案例中的每个操作，
若栈不为空，输出相应的栈中最小元素。否则，输出NULL。

样例输入： 
7
s 3
s 4
s 2
s 1
o
o
s 0
样例输出： 
3
3
2
1
2
3
0
cpp 代码如下： 
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
int main() {
int n,tail,t,opt[10002];
char str[3];
while(scanf("%d",&n) != EOF){
tail=1;opt[0] = 100000000;
while(n--){scanf("%s",str);
if(str[0] == 's'){scanf("%d",&t);opt[tail] = t < opt[tail-1] ? t:opt[tail-1];printf("%d\n",opt[tail]);tail++;
}else{tail--;
if(tail > 1)printf("%d\n",opt[tail-1]);else puts("NULL");}}}return 0;}
