题目来源：2006年浙江大学计算机及软件工程研究生机试真题

题目描述：
    读入一个只包含 +, -, *, / 的非负整数计算表达式，计算该表达式的值。
输入：
    测试输入包含若干测试用例，每个测试用例占一行，每行不超过200个字符，整数和运算符之间用一个空格分隔。没有非法表达式。当一行中只有0时输入结束，相应的结果不要输出。
输出：
    对每个测试用例输出1行，即该表达式的值，精确到小数点后2位。
样例输入： 
1 + 2
4 + 2 * 5 - 7 / 11
0
样例输出： 
3.00
13.36
cpp 代码如下： 
#include "stdio.h"
#include "string.h"
#define MAX 1001

double stack[MAX];
int tail;

int main(){
  int a;
  while(scanf("%d ",&a)&&a!=0){
   tail=0;
   stack[++tail]=1.0*a;//tail始终指向末尾数字位置
   char ch1,ch2;
   while(scanf("%c %d%c",&ch1,&a,&ch2)!=EOF){
    if(ch1=='+'){
     stack[++tail]=1.0*a;//push
    }else if(ch1=='-'){
     stack[++tail]=-1.0*a;//push neg
    }else if(ch1=='*'){
     stack[tail]=stack[tail]*a;//update tail
    }else if(ch1=='/'){
     stack[tail]=stack[tail]/(double)a;//updata tail
    }
    if(ch2!=' ')break; 
  }
  double result=0;
  for(int i=1;i<=tail;i++)result+=stack[i];
  printf("%.2lf\n",result);
  }
return 1;
}
