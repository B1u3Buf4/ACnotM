九度-1066-字符串排序[解题代码]

coder九度OJ, 考研机试围观1322次22 条评论

题目来源：2001年清华大学计算机研究生机试真题(第II套)

题目描述：

 输入一个长度不超过20的字符串，对所输入的字符串，按照ASCII码的大小从小到大进行排序，请输出排序后的结果

输入：

 一个字符串，其长度n<=20

输出：

 输入样例可能有多组，对于每组测试样例，

按照ASCII码的大小对输入的字符串从小到大进行排序，输出排序后的结果
样例输入： 
dcba
样例输出： 
abcd
cpp 代码如下： 
#include <stdio.h>
#include <stdlib.h>
#include <string.h>
char str[21];
int cmp(const void * a,const void *b){
	return *(char *)a - *(char *)b;
}
int main(void) {
	while(scanf("%s",str) != EOF){
		int n = strlen(str);
		qsort(str,n,1,cmp);
		printf("%s\n",str);
	}
	return 0;
}
