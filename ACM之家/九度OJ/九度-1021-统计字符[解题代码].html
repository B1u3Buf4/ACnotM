九度-1021-统计字符[解题代码]

coder九度OJ, 考研机试围观1600次15 条评论

题目来源：2007年浙江大学计算机及软件工程研究生机试真题

题目描述：
    统计一个给定字符串中指定的字符出现的次数。
输入：
    测试输入包含若干测试用例，每个测试用例包含2行，第1行为一个长度不超过5的字符串，第2行为一个长度不超过80的字符串。注意这里的字符串包含空格，即空格也可能是要求被统计的字符之一。当读到’#'时输入结束，相应的结果不要输出。
输出：
    对每个测试用例，统计第1行中字符串的每个字符在第2行字符串中出现的次数，按如下格式输出：
    c0 n0
    c1 n1
    c2 n2
    …
    其中ci是第1行中第i个字符，ni是ci出现的次数。
样例输入： 
I
THIS IS A TEST
i ng
this is a long test string
#
样例输出： 
I 2
i 3
  5
n 2
g 2
cpp 代码如下： 
#include <string.h>
#include <stdio.h>
char str1[10];
char str2[81];
int arr[128];
int i;
int main() {
	while( gets(str1) ){
		if(str1[0] == '#')
			break;
		memset(arr, 0, sizeof(arr));
		gets(str2);
		for(i=0; str2[i]; ++i)
			arr[str2[i]] ++;
		for(i=0; str1[i]; ++i)
			printf("%c %d\n", str1[i], arr[ str1[i]]);

	}
	return 0;
}
