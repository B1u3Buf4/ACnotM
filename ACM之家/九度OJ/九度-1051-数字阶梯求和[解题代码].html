九度-1051-数字阶梯求和[解题代码]

coder九度OJ, 考研机试围观1968次15 条评论

题目来源：2010年哈尔滨工业大学计算机研究生机试真题

题目描述：

给定a和n，计算a+aa+aaa+a…a(n个a)的和。

输入：

测试数据有多组，输入a，n（1<=a<=9,1<=n<=100）。

输出：

对于每组输入,请输出结果。

样例输入： 
1 10
样例输出： 
1234567900
cpp 代码如下： 
#include <stdio.h>
int a,n;
char ans[110];
int main() {
	ans[109] = 0;
	while(scanf("%d %d", &a, &n) != EOF){
		int last = 108,jin = 0,tmp;
		while(n){
			tmp = a*n + jin;
			ans[last] = tmp % 10 + '0';
			jin = tmp /10;
			last--;n--;}
		if(jin > 0) ans[last--] = jin + '0';
		printf("%s\n", ans+last+1);
	}
	return 0;
}
