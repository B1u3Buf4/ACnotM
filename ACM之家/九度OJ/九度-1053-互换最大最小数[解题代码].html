九度-1053-互换最大最小数[解题代码]

题目来源：2010年哈尔滨工业大学计算机研究生机试真题

题目描述：

输入一个数n，然后输入n个数值各不相同，调换数组中最大和最小的两个数，然后输出。

输入：

测试数据有多组，输入n(1<=n<=20)，接着输入n个数。

输出：

对于每组输入,输出交换后的结果。

样例输入： 
2
1 3
样例输出： 
3 1
cpp 代码如下： 
#include<stdio.h>
int main()
{
               int n,a[20],i,max,min,x,y;
        while(scanf("%d",&n)!=EOF)
        {
                for(i=0;i<n;i++)
                {
                        scanf("%d",&a[i]);
                        min=max=a[0];
                        x=y=0;
                }
                for(i=0;i<n;i++)
                {   
                        if(a[i]<min)
                        {
                                                min=a[i];
                                        x=i;
                        }
                        if(a[i]>max)
                        {
                                max=a[i];
                                y=i;
                        }
                }
                a[y]=min;
                                a[x]=max;
                                for (i=0;i<n;i++)
                {
                                             printf("%d",a[i]);
                             if(i!=n-1) 
                             printf(" ");
                                             else 
                            printf("\n");
                }
        }
    return 0;
}
