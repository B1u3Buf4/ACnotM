九度-1049-字符串去特定字符[解题代码]

coder九度OJ, 考研机试围观893次15 条评论

题目来源：2009年哈尔滨工业大学计算机研究生机试真题

题目描述：

输入字符串s和字符c，要求去掉s中所有的c字符，并输出结果。

输入：

测试数据有多组，每组输入字符串s和字符c。

输出：

对于每组输入,输出去除c字符后的结果。

样例输入： 
heallo
a
样例输出： 
hello
java 代码如下： 
import java.util.Scanner;
public class Main {
	public static void main(String[] args) {
		Scanner s = new Scanner(System.in);
		while(s.hasNext())
			System.out.println(s.next().replaceAll(s.next(), ""));
	}
}
