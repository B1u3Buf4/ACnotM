九度-1098-字母统计[解题代码]

coder九度OJ, 考研机试围观691次28 条评论

题目来源：2009年上海交通大学计算机研究生机试真题

题目描述：

输入一行字符串，计算其中A-Z大写字母出现的次数

输入：

案例可能有多组，每个案例输入为一行字符串。

输出：

对每个案例按A-Z的顺序输出其中大写字母出现的次数。

样例输入： 
DFJEIWFNQLEF0395823048+_+JDLSFJDLSJFKK
样例输出： 
A:0
B:0
C:0
D:3
E:2
F:5
G:0
H:0
I:1
J:4
K:2
L:3
M:0
N:1
O:0
P:0
Q:1
R:0
S:2
T:0
U:0
V:0
W:1
X:0
Y:0
Z:0
cpp 代码如下： 
#include <stdio.h>
#include <string.h>
char str[1000];
int ans[128];
int main(){
	while(gets(str)){
		int len = strlen(str);
		for(int i='A'; i<='Z'; i++)
			ans[i] = 0;
		for(int i=0; i<len; i++){
			ans[str[i]]++;
		}
		for(int i='A'; i<='Z'; i++)
			printf("%c:%d\n",i,ans[i]);
	}
	return 0;
}
