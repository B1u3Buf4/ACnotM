九度-1152-点菜问题[解题代码]

coder九度OJ, 考研机试围观1551次24 条评论

题目来源：2010年北京大学计算机研究生机试真题

题目描述：

    北大网络实验室经常有活动需要叫外买，但是每次叫外买的报销经费的总额最大为C元，有N种菜可以点，经过长时间的点菜，网络实验室对于每种菜i都有一个量化的评价分数（表示这个菜可口程度），为Vi，每种菜的价格为Pi, 问如何选择各种菜，使得在报销额度范围内能使点到的菜的总评价分数最大。
    注意：由于需要营养多样化，每种菜只能点一次。

输入：

    输入的第一行有两个整数C（1 <= C <= 1000）和N（1 <= N <= 100），C代表总共能够报销的额度，N>代表能点菜的数目。接下来的N行每行包括两个在1到100之间（包括1和100）的的整数，分别表示菜的>价格和菜的评价分数。

输出：

    输出只包括一行，这一行只包含一个整数，表示在报销额度范围内，所点的菜得到的最大评价分数。

样例输入： 
90 4
20 25
30 20
40 50
10 18
40 2
25 30
10 8
样例输出： 
95
38
cpp 代码如下： 
#include <iostream>
#include <string.h>

using namespace std;
int cost[101];
int val[101];
int opt[1001];

int max2(int a,int b){
	return a > b ? a : b;
}
int main() {
	int max,n;
	while(cin >> max >> n){
		memset(opt,0,sizeof(opt));
		for(int i=0; i<n; i++)
		{
			cin >> cost[i] >> val[i];
		}

		for(int i=0; i<n; i++)
			for(int j=max; j>=cost[i]; j--){

					opt[j] = max2( opt[j - cost[i]] + val[i], opt[j] );
			}

		cout << opt[max] << endl;
	}
	return 0;
}
