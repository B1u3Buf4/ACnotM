九度-1068-球的半径和体积

coder九度OJ, 考研机试围观1739次19 条评论

题目来源：2003年清华大学计算机研究生机试真题

题目描述：
输入球的中心点和球上某一点的坐标，计算球的半径和体积
输入：
球的中心点和球上某一点的坐标，以如下形式输入：x0 y0 z0 x1 y1 z1
输出：
输入可能有多组，对于每组输入，输出球的半径和体积，并且结果保留三位小数
样例输入：
0 0 0 1 1 1
样例输出：
1.732 21.766
提示：
为避免精度问题，PI值请使用arccos(-1)。 
#include<stdio.h>
#include<stdlib.h>
#include<math.h>
int main()
{
        double  r,x0,y0,z0,x1,y1,z1;
  while(scanf("%lf%lf%lf%lf%lf%lf",&x0,&y0,&z0,&x1,&y1,&z1)!=-1)
    {

       r=sqrt(pow((x0-x1),2)+pow((y0-y1),2)+pow((z0-z1),2));
       printf("%.3lf %.3lf\n",r,pow(r,3)*4*acos(-1)/3);
     }
   return 0;
}
