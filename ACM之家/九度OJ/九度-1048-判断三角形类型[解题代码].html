九度-1048-判断三角形类型[解题代码]

coder九度OJ, 考研机试围观1682次15 条评论

题目来源：2009年哈尔滨工业大学计算机研究生机试真题

题目描述：

给定三角形的三条边，a,b,c。判断该三角形类型。

输入：

测试数据有多组，每组输入三角形的三条边。

输出：

对于每组输入,输出直角三角形、锐角三角形、或是钝角三角形。

样例输入： 
3 4 5
样例输出： 
直角三角形
cpp 代码如下： 
#include <iostream>
using namespace std;

int a,b,c,t;
int main() {
	while(cin >> a >> b >> c){
		if(a > b)
			swap(a,b);
		if(a>c)
			swap(a,c);
		if(b>c)
			swap(b,c);
		t = a*a + b*b - c*c;
		if(t==0)
			cout << "直角三角形\n" ;
		else if(t > 0){
			cout << "锐角三角形\n" ;
		}else
			cout << "钝角三角形\n";
	}
	return 0;
}
