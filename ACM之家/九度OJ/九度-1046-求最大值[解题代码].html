九度-1046-求最大值[解题代码]

coder九度OJ, 考研机试围观1095次10 条评论

题目来源：2009年哈尔滨工业大学计算机研究生机试真题

题目描述：

输入10个数，要求输出其中的最大值。

输入：

测试数据有多组，每组10个数。

输出：

对于每组输入,请输出其最大值（有回车）。

样例输入： 
10 22 23 152 65 79 85 96 32 1
样例输出： 
max=152
cpp 代码如下： 
#include <stdio.h>
int main(){
	int max,tmp,i;
	while(scanf("%d",&tmp) != EOF){
		max = tmp;
		for(i=0; i<9; i++){
				scanf("%d", &tmp);
				if(tmp > max)
					max = tmp;
			}
		printf("max=%d\n",max);
	}
}
