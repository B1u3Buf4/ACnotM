九度-1104-整除问题[解题代码]

coder九度OJ, 考研机试围观1944次26 条评论

题目来源：2011年上海交通大学计算机研究生机试真题

题目描述：

给定n，a求最大的k，使n！可以被a^k整除但不能被a^(k+1)整除。

输入：

两个整数n(2<=n<=1000)，a(2<=a<=1000)

输出：

一个整数.

样例输入： 
6 10
样例输出： 
1
cpp 代码如下： 
#include <stdio.h>
int main() {
	int n, a;
	while (~scanf("%d %d", &n, &a)) {
		int cnt = 0;
		int sum = 1;
		int i;
		for (i = n; i >= 1; i--) {
			sum *= i;
			while (sum % a == 0) {cnt++;sum /= a;}
			sum %= a;
		}
		printf("%d\n", cnt);
	}
	return 0;
}
