九度-1194-八进制[解题代码]

题目来源：2008年华中科技大学计算机研究生机试真题

题目描述：

输入一个整数，将其转换成八进制数输出。

输入：

输入包括一个整数N(0<=N<=100000)。

输出：

可能有多组测试数据，对于每组数据，
输出N的八进制表示数。

样例输入： 
7
8
9
样例输出： 
7
10
11
java 代码如下： 
import java.util.Scanner;


public class Main{

	public static void main(String[] args) {
		Scanner s = new Scanner(System.in);
		while(s.hasNextInt()){
			
			System.out.printf("%o\n",s.nextInt());
		}
	}

}
