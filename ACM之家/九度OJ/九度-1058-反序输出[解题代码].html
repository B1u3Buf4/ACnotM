九度-1058-反序输出[解题代码]

题目来源：2000年清华大学计算机研究生机试真题

题目描述：

输入任意4个字符(如：abcd)， 并按反序输出(如：dcba)

输入：

题目可能包含多组用例，每组用例占一行，包含4个任意的字符。

输出：

对于每组输入,请输出一行反序后的字符串。
具体可见样例。

样例输入： 
Upin
cvYj
WJpw
cXOA
样例输出： 
nipU
jYvc
wpJW
AOXc
cpp 代码如下： 
#include <stdio.h>
#include <string.h>
int main()
{
    char temp;
    char a[5];
    int i;
    while(gets(a))
    {
        for(i=0;i<2;i++)
        {
            temp=a[i];
            a[i]=a[3-i];
            a[3-i]=temp;
         }
        puts(a);
    }
    return (0);
}
