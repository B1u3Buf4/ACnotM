九度-1056-最大公约数[解题代码]

coder九度OJ, 考研机试围观646次15 条评论

题目来源：2011年哈尔滨工业大学计算机研究生机试真题

题目描述：

输入两个正整数，求其最大公约数。

输入：

测试数据有多组，每组输入两个正整数。

输出：

对于每组输入,请输出其最大公约数。

样例输入： 
49 14
样例输出： 
7
java 代码如下： 
import java.util.Scanner;

public class Main {
	static int a,b;
	static int f(int x,int y){
		
		while(y != 0){
			int tmp = y;
			y = x % y;
			x = tmp;
		}
		return x;
	}
	public static void main(String[] args) {
		Scanner s = new Scanner(System.in);
		while(s.hasNextInt()){
			a = s.nextInt();
			b = s.nextInt();
			
			if(a < b){
				System.out.println(f(b,a));
			}else
				System.out.println(f(a,b));
			
		}
	}

}
