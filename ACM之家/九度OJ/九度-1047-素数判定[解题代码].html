九度-1047-素数判定[解题代码]

题目来源：2009年哈尔滨工业大学计算机研究生机试真题

题目描述：

给定一个数n，要求判断其是否为素数（0,1，负数都是非素数）。

输入：

测试数据有多组，每组输入一个数n。

输出：

对于每组输入,若是素数则输出yes，否则输入no。

样例输入： 
13
样例输出： 
yes
cpp 代码如下： 
//简单的素数判断.....

#include <iostream>
#include <cmath>

using namespace std;
int N;

void solve()
{
        if(N<=1){cout<<"no"<<endl;return;}
        else 
        {
                int M =(int)sqrt((double)N);
                for(int i = 2; i<=M; i++)
                        if(N%i == 0)
                        {
                                cout<<"no"<<endl;
                                return;
                        }
        }
        cout<<"yes"<<endl;
}

int main()
{

        while(cin>>N)
        {
                solve();
        }
        return 0;
}
