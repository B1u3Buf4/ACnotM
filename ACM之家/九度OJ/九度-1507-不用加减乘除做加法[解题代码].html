题目描述：

写一个函数，求两个整数之和，要求在函数体内不得使用+、-、*、/四则运算符号。

输入：

输入可能包含多个测试样例。
对于每个测试案例，输入为两个整数m和n(1<=m,n<=1000000)。

输出：

对应每个测试案例，输出m+n的值。

样例输入： 
3 4
7 9
样例输出： 
7
16
cpp 代码如下： 
#include <iostream>
using namespace std;
int m,n;
//例如 100 11 = 111， 直接异或即可。终止条件即为 100 & 011 = 0
int fadd(int a,int b){
	int ans,up;
	int num1 = a;
	int num2 = b;
	do{
		ans = num1 ^ num2;
		up = (num1 & num2) << 1;
		num1 = ans;
		num2 = up;
	}while(up != 0);
	return ans;
}

int main() {
	while(cin >> m >> n)
		cout << fadd(m,n) << endl;

	return 0;
}
