## 771 宝石与石头

水题。

## python resolvent 1

    class Solution:
        def numJewelsInStones(self, J, S):
            """
            :type J: str
            :type S: str
            :rtype: int
            """
            con = 0
            for i in S:
                if i in J:
                    con += 1
            return con
